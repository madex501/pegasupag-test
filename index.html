Funciones anti-raid

Estos sistemas son ideales para mantener protegido tu servidor de cualqueir amenaza 💪

1anti joins

El sistema anti joins es un sistema creado para servidores privados, este evita que algun usuario/bot se una 👥

2anti bots

El sistema anti bots fue creado para aquellos servidores que quieren protegerse de los bots raiders, este sistema evita que algun bot se una 🤖

3anti links

El sistema anti links evita que los usuarios del servidor envien algun tipo de link 🔗

4anti linksMaliciosos

El sistema anti linksMaliciosos envita que los usuarios de un servidor envien links maliciosos 🔗💥

5anti pings

El sistema anti pings evita que los usuarios del servidor mencionen algun rol 🏅

6anti borrarCanales

El sistema anti borrarCanales evita que algun usuario borre mas de 5 canales en 10 segundos 🌀

7anti crearCanales

El sistema anti crearCanales evita que algun usuario cree mas de 5 canales en 10 segundos ⚡

8logs

El sistema de logs (anti-raid) tiene la funcion de recibir todas los registros de los sistemas anti-raid activados en el servidor 🚀
